Components


  • Bare Metal Host: This provides a standard interface for managing bare metal host in a kubernetes way, it basically acts as a bridge between physical hosta and kubernetes.

  • Cluster API:  Provides a way to manage and deploy kubernetes on bare metal infrastructure.

  • Metal3-io/Ironic operator: Ironic is a kubernetes operator that manages openstack bare metal operating software installation systems,issued by Capi-metal3 to provide and manage bare metal hosts.

  • Metal3 Deployment Kit: Set of tools that make it feasible to process the development and testing of metal3 components on any local machine or in a virtual environment.

    • All these components form together a framework for the management of bare metal in a kubernetes way in order to let the users and organizations take the advantage and benefit from it.
