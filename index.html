We are a Cloud Native Computing Foundation sandbox project

Metal3 provides a powerful, flexible platform for managing bare metal infrastructure using Kubernetes. It allows users to treat bare metal nodes in the same way as virtual machines or cloud instances, providing a level of abstraction that makes it easier to manage and scale bare metal infrastructure.




FAQ


  • why is metal3 important?

    Metal3 is important because of its role in industry, economics, the environment, and technological advancement.


  • what is bare metal ?

    Bare metal refers to a computer system or a device that runs directly on the underlying hardware without any intervening operating system or software layer. This means that the software is directly installed onto the physical machine or device, rather than running on top of an operating system like Windows or Linux.


  • What is an API ?

    API stands for Application Programming Interface. An API is a set of protocols, routines, and tools that enable different software applications to communicate with each other.


  • what is kubernetes cluster?

    A Kubernetes cluster is a group of nodes (physical or virtual machines) that run containerized applications managed by Kubernetes.
